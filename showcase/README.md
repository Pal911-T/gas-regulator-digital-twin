# 睿轴驭燃 v3.3 主展示基线

本目录作为后续 GitHub Pages 主展示版本。

## 基线决策

- 保留 v3.3 的工业视觉、顶部导航、卡片、趋势图、告警弹窗、设备管理和工程评估结构。
- 将多设备通用调压器业务重构为 RTJ-25/4.0Z 双级轴流式调压器数字孪生。
- 当前阶段仅实现前端功能与底层逻辑展示，不接入真实 5G、MQTT、数据库或 PLC。
- 所有运行值必须标明数据性质：仿真值、设定值、计算值、预测值。

## 第一轮改造范围

1. 设备对象改为 RTJ-25/4.0Z-001。
2. 遥测字段改为 P1、Pm、P2、X1、X2、一级/二级开度、Qn、T、设定压力和状态量。
3. 增加采集链路页面：外部传感器 → 智能阀位指示器/边缘终端 → 5G → MQTT → 时序库 → 告警与分析。
4. 增加正常、流量阶跃、进口压力波动、出口超压、出口欠压、一级/二级卡涩、通信中断、传感器异常等工况。
5. 增加精度试验、预测预警、告警处置和数据来源说明。

原始 v3.3 单文件 HTML 将迁移为 `showcase/index.html`，随后再逐步拆分为可维护模块。
